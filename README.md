# Paper

This folder contains the technical note / preprint-style PDF for the Union3 low-z / mid-z nuisance robustness diagnostic, together with its LaTeX source and figures.

## Included files

- `union3_nuisance.pdf` — current technical note / preprint-style PDF (13 pages)
- `union3_nuisance.tex` — LaTeX source
- `fullplanck_w0wa_desi.pdf` — full-likelihood w0–wa contour figure (Figure 1)
- `overlap_angle.pdf` / `overlap_angle.png` — template–CPL overlap figure (Figure 2)

## Note

This document describes compressed-data robustness diagnostics, a public-data Cobaya cross-check, and a full-likelihood cross-check (Planck NPIPE CamSpec + DESI DR1 BAO + Union3) in which the nuisance amplitude is sampled directly. It does not claim to reproduce the official, internally validated DESI/CMB/Union3 systematic treatment.

The GitHub repository provides the reproduction scripts, expected outputs, and data-placement instructions for checking the main results.
