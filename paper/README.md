# Paper

This folder contains the technical note / preprint-style PDF for the Union3 low-z / mid-z nuisance robustness diagnostic.

## Included file

- `Union3_nuisance_preprint_Hass 2026.pdf`

## Note

This document describes a compressed-data robustness diagnostic. It does not claim to reproduce the official DESI/CMB/Union3 full likelihood.

The GitHub repository is intended to provide the reproduction scripts, expected outputs, and data-placement instructions for checking the main results.
