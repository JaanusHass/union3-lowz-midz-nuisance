# Data

This project uses public Union3 and Union3.1 data products from the rubind/union3_release repository.
The original data files are not redistributed here; download them from the public Union3 release.

## Required by the compressed-data scripts (01-08)

Place these FITS files in this folder before running the scripts:

- mu_mat_union3_cosmo=2_mu.fits  (primary; used by scripts 01-04, 06-08)
- mu_mat_union3.1_UNITY1.7_template_cosmo=2_0_mu.fits  (used by script 05)
- mu_mat_union3.1_UNITY1.8_template_cosmo=2_0_mu.fits  (used by script 05)

## Referenced by the note / public Union3 release, but NOT required by scripts 01-08

- lcfit_Union3.tar.gz
