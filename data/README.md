# Data

This project uses public Union3 and Union3.1 data products from the rubind/union3_release repository.

Required public files:

- mu_mat_union3_cosmo=2_mu.fits
- lcfit_Union3.tar.gz
- Union3.1 UNITY1.7 compressed matrix
- Union3.1 UNITY1.8 compressed matrix

The original data files are not redistributed here. Users should download them from the public Union3 release repository.
